Movie Search Engine

NODE.JS, EXPRESS, SQL, MOCHA CHAI, BOOTSTRAP, HEROKU, ENCRYPTION, TEMPLATING, RELATIONAL DATABASE DESIGN

Movie Search Engine: A Full Stack web application. With encrypted session cookies the user creates an account whose password has been hashed and stored in a SQL database. The user then searches a movie title which, through the web server, queries IMBD.com and scrapes the results which are sent back to the client and displayed on the same page. User searches are also stored in the database and are available to display.
